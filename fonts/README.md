# Fonts

The theme is designed to pair with **Cascadia Code** (Microsoft), a humanist monospaced font with rounded forms that work well with the cream-and-purple palette.

The installer can install Cascadia Code automatically on Debian-based systems (Kali, Ubuntu, Debian) via:

```
sudo apt install -y fonts-cascadia-code
```

This installs the entire Cascadia family:

- **Cascadia Code** — the main font, with ligatures
- **Cascadia Mono** — same shapes, no ligatures
- **Cascadia Code NF** — Nerd Font variant (10k+ icons for Starship, Powerlevel10k, etc.)
- **Cascadia Code PL** — Powerline glyphs

Available styles: Light, SemiLight, Regular, SemiBold, Bold (each with italic).

## Alternatives

If you prefer a different style, any of these work well with the Exclusive palette:

- **Commit Mono** — the most humanist option, closest to Lato in spirit
- **JetBrains Mono** — more geometric and vertical
- **Fira Code** — classic programmer font with ligatures
- **Iosevka Aile** — narrow, humanist, very rounded
